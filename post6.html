Lập trình mạng với Java – Hiểu rõ Socket và mô hình Client-Server

Từ khóa: Java, lập trình mạng, socket, client-server, TCP, UDP

1. Giới thiệu

Socket là cầu nối giữa hai thiết bị giao tiếp qua mạng. Trong Java, bạn có thể sử dụng các lớp như ServerSocket và Socket để xây dựng ứng dụng mạng theo mô hình client-server. Đây là nền tảng cho các ứng dụng như chat, game online, hoặc hệ thống giám sát.

2. Kiến thức nền tảng

  • Socket: điểm cuối của kết nối hai chiều giữa hai tiến trình.
  • ServerSocket: lắng nghe yêu cầu kết nối từ client.
  • TCP: truyền dữ liệu tin cậy, kiểm tra lỗi.
  • UDP: truyền nhanh, không đảm bảo độ tin cậy.

3. Ví dụ Java – Tạo Server đơn giản


// Server.java
import java.net.*;
import java.io.*;

public class Server {
    public static void main(String[] args) throws IOException {
        ServerSocket server = new ServerSocket(5000);
        System.out.println("Server đang chờ kết nối...");
        Socket socket = server.accept();
        BufferedReader input = new BufferedReader(new InputStreamReader(socket.getInputStream()));
        PrintWriter output = new PrintWriter(socket.getOutputStream(), true);
        output.println("Kết nối thành công!");
        System.out.println("Client nói: " + input.readLine());
        server.close();
    }
}

4. Ứng dụng thực tế

  • Ứng dụng chat nội bộ trong mạng LAN.
  • Game online với kết nối thời gian thực.
  • Hệ thống giám sát thiết bị từ xa.

5. Kết luận

Socket là nền tảng quan trọng cho mọi ứng dụng mạng. Việc hiểu rõ cách hoạt động của mô hình client-server và cách sử dụng socket trong Java sẽ giúp bạn xây dựng những hệ thống kết nối mạnh mẽ, tin cậy và hiệu quả.
